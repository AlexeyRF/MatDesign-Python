## Быстрый старт

Импортируйте модуль в свой скрипт и вызовите нужные функции. 

```python
import matui

# Получаем 5 доминирующих цветов (от светлого к темному)
colors = matui.get_materials(5)
print(colors) 
# Пример вывода: [(240, 240, 240), (180, 150, 120), (50, 50, 50), ...]

```

### `get_materials(color_count: int) -> list[tuple]`

Находит обои, извлекает заданное количество доминирующих цветов и сортирует их по уровню яркости.

* **Аргументы:**
* `color_count` — количество цветов, которое нужно извлечь.

* **Возвращает:** Список кортежей `(R, G, B)`, отсортированный от светлого цвета к темному. Яркость определяется расстоянием от черного цвета `(0, 0, 0)`.
* **Исключения:** Возвращает `FileNotFoundError`, если обои не найдены.

### `get_safe_materials(color_count: int, min_distance: float) -> list[tuple] | bool`

Генерирует отсортированную палитру и проверяет, достаточен ли контраст между самым светлым и самым темным цветами изображения.

* **Аргументы:**
* `color_count` — количество извлекаемых цветов.
* `min_distance` — минимально допустимое расстояние между первым (самым светлым) и последним (самым темным) цветами палитры (максимум 441).

* **Возвращает:** Список цветов `[(R, G, B), ...]`, если расстояние больше или равно `min_distance`. Возвращает `False`, если палитра недостаточно контрастна.

### `is_available(min_distance: float) -> int`

Анализирует все пиксели изображения и подсчитывает количество уникальных цветов, которые находятся друг от друга на расстояние `>= min_distance`.

* **Аргументы:**
* `min_distance` — минимальное расстояние между любыми двумя цветами в итоговом списке.

* **Возвращает:** Количество найденных контрастных цветов. Если на фото удалось найти менее двух таких цветов (например, изображение однотонное), функция вернет `False`.

### `color_distance(c1: tuple, c2: tuple) -> float`

Вспомогательная функция для вычисления евклидова расстояния между двумя цветами.

* **Аргументы:** `c1`, `c2` — кортежи формата `(R, G, B)`.
* **Возвращает:** Дистанцию в виде числа с плавающей точкой (от `0.0` до `~441.67`).

## Пример 

```python
from matui import get_materials, get_safe_materials, is_available

def main():
    # 1. Простое получение цветов
    print("Получение 5 цветов:")
    colors = get_materials(5)
    for i, color in enumerate(colors, 1):
        print(f"{i}. RGB: {color}")
        
    # 2. Получение "безопасной" палитры с проверкой контраста
    print("\nЗапрос безопасной палитры (мин. дистанция 150):")
    safe_colors = get_safe_materials(5, 150)
    if safe_colors:
        print("Контраст в норме. Палитра одобрена.")
    else:
        print("Обои слишком тусклые/однотонные, контраст не прошел проверку.")

    # 3. Анализ вариативности цветов на изображении
    print("\nАнализ уникальных контрастных цветов на обоях (мин. дистанция 100):")
    count = is_available(100)
    if count == 0:
        print("На обоях нет и 2 цветов, достаточно удаленных друг от друга.")
    else:
        print(f"Найдено сильно отличающихся цветов: {count}")

if __name__ == "__main__":
    main()

```
